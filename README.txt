Brothers FIFA Android App

How to open/build:
1. Extract this folder.
2. Open Android Studio.
3. Click File > Open and select the BrothersFifaAndroid folder.
4. Let Gradle sync.
5. Click Build > Build Bundle(s) / APK(s) > Build APK(s).
6. The APK will be in app/build/outputs/apk/debug/app-debug.apk.

What this app is:
- A native Android WebView wrapper for your FIFA 25 tournament page.
- It loads app/src/main/assets/index.html.
- Firebase Realtime Database sync is already built into the HTML.

Firebase URL:
https://brothers-fifa-default-rtdb.firebaseio.com

Realtime Database path:
fifa25TournamentHub/shared
